# Markdown Dokumentation

## Fordele ved at bruge Markdown language
* Fordele ved Markdown (.md)
* Let at læse både som kode og færdigt dokument
* Understøtter overskrifter, lister, links, billeder kodeblokke osv.
* Bruges som standard i mange udvikler- og AI-værktøjer
Perfekt til dokumentation og noter
* Kan konverteres til HTML, PDF osv.
* Det bliver vist pænt og struktureret på fx:
    * GitHub
    * Obsidian
    * Notion
    * dokumentation
    * README-filer
    * AI-projekter
